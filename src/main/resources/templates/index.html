Ayuda

Esta aplicación permite gestionar una biblioteca. Funciones principales:

  • Filtra elementos por tipo (Libros, Revistas, DVDs) usando el menú desplegable.
  • Busca por género, autor o editorial usando los formularios de búsqueda.
  • Agrega nuevos elementos haciendo clic en 'Agregar Nuevo Elemento'.
  • Edita o elimina elementos desde la lista.
